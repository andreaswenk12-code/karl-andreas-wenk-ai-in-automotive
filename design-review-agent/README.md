# AI Design Review Agent for Automotive Engineering (BEV/ICE)

**n8n workflow** built by Karl Andreas Wenk (FEV China / VW Group)

## What it does
Analyzes part drawings, specifications, or descriptions and automatically delivers a professional design review:
- ✅ Part Summary
- ❌ Issues Found (with severity levels)
- 🔧 Recommended Actions (Catia-compatible)
- 📊 Regulatory / VW Compliance Check
- 💡 Additional Opportunities (weight saving, cost reduction, manufacturability)

## Why it matters for me
With 14+ years in automotive design, project management and BEV/ICE technologies at FEV China, I built this agent to replicate the kind of design review I used to lead for Volkswagen Group projects — but now fully automated.

## How to use
1. Import `design-review-agent.json` into your own n8n (self-hosted Community Edition)
2. Add your OpenAI API key
3. Paste any part spec → get instant expert feedback

## Tech
- n8n Community Edition (self-hosted)
- OpenAI `gpt-4o` (or any compatible LLM)
- Fully customizable system prompt
